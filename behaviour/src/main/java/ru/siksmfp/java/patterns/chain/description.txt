Паттер для групповой обработки объектов. 
У нас есть файлы разного формата - текстовые. звуковые, видео. Их нужно
по-особому обрабатывать. Для этой обработки проводим файл через обработчиков
файлов. Тот обработчик, который сможет работать с файлов и выполнит работу,
остальные не сделаю с ним ничего.
